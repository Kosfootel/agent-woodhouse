# Vigil Home

**Your home's guardian.**

An intelligent security appliance that protects your smart home, learns what's normal, and speaks only when it matters.

---

## Product Vision

Vigil Home is a dedicated hardware appliance that provides continuous, autonomous security monitoring for residential smart homes. Unlike traditional antivirus or firewall solutions, Vigil operates as a "guardian" — a persistent presence that understands your devices, establishes trust baselines, and communicates contextually about security and privacy concerns.

### Key Differentiators

- **Guardian Persona:** Not a tool you use, but a presence that watches
- **Device Trust Scoring:** Every device has a trust score that evolves over time
- **Narrative Communication:** Contextual explanations, not alerts
- **CleanSL8 Integration:** Seamless device transfer when you move
- **Privacy-First:** Local processing, minimal cloud dependency
